Firebird System Requirements

Database: The minimal Firebird version is 2.5.0.

OS: Windows XP,Windows Vista,Windows Server 2003,Windows Server 2008.

Firebird Installation

1.Install & configure Firebird database & server

Download archive from here, choose your architecture.

Files for server called like:

For win32: Firebird-2.5.0.23247_1_Win32_Beta1.exe

For win64: Firebird-2.5.0.23247_1_Win64_Beta1.exe

Files for embedded called like: Firebird-2.5.0.23247-1_Win32_embed_Beta1.zip

For further instruction see firebird official site

1.1.Install Firebird server

 


1. Download the archive or version from our svn.


2. Unzip the entire archive to to a folder you want to run YAF in on your web server.
Apply all svn patches to corresponding svn version if present



STEP 2. INSTALL/CHECK Firebird DB SETTINGS.

STEP 3. REPLACE FILES and folders in the archive.


Replace all the files in the archive. If a folder misses, create it.


STEP 4a. REPLACE OR ADD FILES IN FOLDERS.

Simply replace YAF distribution files with /Distribution folder content or add new ones if they are in distribution.

STEP 4b. COMPILE YAF(optional).

Firebird Configuration



1.Page size and encoding



If you use UTF8 keep in mind that page size must be more 4096kB.



2.Connection strings examples



-- firebird as embeded minimal --



connectionString="Database=C:\Documents and Settings\bob\My Documents\Projects\yaffirebird\YetAnotherForum.NET\App_Data\yafnet.fdb; Server Type=1"



-- firebird as embeded full or as a file --



connectionString="User=SYSDBA;Password=myfirebird; Database=C:\Documents and Settings\bob\My Documents\Projects\yaffirebird\YetAnotherForum.NET\App_Data\yafnet.fdb;Dialect=3; Charset=UNICODE_FSS;Role=; Connection lifetime=0;Connection timeout=15;Pooling=True; Server Type=1"



-- firebird as server minimal --

connectionString="Server=localhost; User=SYSDBA;Password=masterkey;Database=C:\data\test.fdb"



-- firebird as server full
